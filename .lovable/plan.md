

## HSK Grammar Reference Website — 中文语法

A Chinese-inspired learning resource with all essential HSK 1–6 grammar points, featuring a global search and per-level pages.

### Design & Theme
- **Chinese-inspired aesthetic**: Rich red and gold accents, subtle traditional patterns/borders, calligraphy-style heading fonts
- **Dark background with warm tones** for an elegant scholarly feel
- **Color-coded HSK levels** (HSK1 = green, HSK2 = blue, HSK3 = purple, HSK4 = orange, HSK5 = red, HSK6 = gold)

### Pages & Navigation
- **Home page** with a hero section, global search bar, and quick links to all 6 HSK levels
- **6 HSK level pages** (HSK 1–6), each listing all grammar points for that level
- **Top navigation bar** with links to each level + search

### Grammar Point Card
Each grammar point displays:
- **Grammar pattern** (Chinese characters)
- **English name/explanation**
- **Structure** (e.g., Subject + 把 + Object + Verb)
- **2–3 example sentences** with Chinese, pinyin, and English translation
- **HSK level badge**

### Search & Filter
- **Global search bar** on home and all pages — search by Chinese pattern, English keyword, or pinyin
- **Category filter** (e.g., particles, conjunctions, comparison, time expressions)
- Search results show matching grammar cards with their HSK level highlighted

### Content
- Hardcoded grammar data covering ~30–50 key grammar points per HSK level (180–300 total)
- Organized in a structured data file for easy future updates
- Categories assigned to each grammar point for filtering

### Key Interactions
- Click a grammar card to expand full details
- Smooth scrolling and accordion-style expansion
- Responsive design for mobile student access
